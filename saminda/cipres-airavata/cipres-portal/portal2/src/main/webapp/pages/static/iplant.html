The iPlant Collaborative is a large NSF-funded project that provides users with access to a wide variety of software tools and capabilities, as well as access to a large number of biological databases. Users can access these resources through the iPlant Discovery Environment (DE), a flexible and innovative web client. The login tab presented here allows users who are registered with the iPlant Collaborative to work both in the iPlant DE, and in the CIPRES Science Gateway, through a single-login mechanism. We encourage our users to visit the iPlant DE page, and to register and test drive the application. If you have further questions, please let us know.
